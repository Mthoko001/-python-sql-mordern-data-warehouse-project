# 🏗️ Modern Data Warehouse — Sales ETL Pipeline

A production-style end-to-end ETL pipeline that extracts raw sales data, performs data quality checks and transformations, and loads curated data into a SQL Server data warehouse (Bronze layer).

This project demonstrates real-world data engineering practices including bulk loading, data validation, modular architecture, and SQL Server integration.

---

## 🚀 Project Overview

The pipeline performs the following:

- **Extract** raw sales data from source files  
- **Transform** data (cleaning, deduplication, type fixes, calculations)  
- **Load** data efficiently into SQL Server Bronze layer  
- **Validate** data quality and log statistics  
- **Optimize** inserts using `fast_executemany` bulk loading
